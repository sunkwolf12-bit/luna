# Guía de Facturación - Nelly (Turno Nocturno)

Esta guía establece el procedimiento estándar para la facturación de depósitos y transferencias, asegurando el cumplimiento con las normativas del SAT.

## 1. Proceso de Identificación
- Localizar el **Folio** del cliente en el archivo de Excel proporcionado.
- Buscar dicho folio en el sistema **SIGA**.

## 2. Determinación de Datos Fiscales
### Escenario A: El cliente tiene "Cédula Fiscal" en SIGA
- Descargar el PDF en el apartado de "Documentación".
- Facturar **estrictamente** a la Razón Social y RFC que aparecen en la Constancia de Situación Fiscal (aunque el nombre en SIGA sea distinto, p. ej., un chofer).
- **Uso de CFDI:** G03 (Gastos en General).
- **Régimen Fiscal:** El que indique su constancia.

### Escenario B: El cliente NO tiene datos fiscales (Público en General)
- Utilizar el **RFC Genérico: XAXX010101000**.
- **Régimen Fiscal:** 616 (Sin obligaciones fiscales).
- **Uso de CFDI:** S01 (Sin efectos fiscales).
- **Fundamento:** Se utiliza para personas sin actividad económica o público en general. No tiene efectos de deducción para el receptor, pero es obligatorio para nuestro control de ingresos.

## 3. Forma de Pago (Tabla de Referencia)
Consultar siempre la tabla de bancos para asignar la clave correcta:
- **01 (Efectivo):** Farmacias GDL, 7-Eleven, Banorte (sin nombre).
- **03 (Transferencia):** BBVA, Santander, Bancoppel, Banamex, HSBC, etc.
- **04 (Tarjeta):** Terminal Banorte.

## 4. Consideraciones Importantes
- **IVA:** El servicio no causa IVA. No realizar desgloses manuales.
- **Dudas:** Si los datos de la Cédula Fiscal parecen inconsistentes o el RFC no valida, consultar a la Gerencia de Cobranza antes de timbrar.
