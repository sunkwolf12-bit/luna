# Mensaje para Cobradores - Notificación de Entrega de Pólizas

---

**Buenos días equipo,**

Les escribo para comunicarles un nuevo requerimiento que entra en vigor a partir de hoy:

**A partir de ahora, cuando realicen una entrega de póliza, deben notificarla de inmediato al Telegram de Cobranza.**

El objetivo es llevar un mejor control de las entregas y evitar que se nos pase dar seguimiento a ninguna póliza.

Les agradezco su cooperación. Cualquier duda, con gusto la resolvemos.

Saludos,
